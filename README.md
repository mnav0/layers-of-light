# layers-of-light

_Repository to be updated with clustering code_

Using the [Global Nighttime Radiance data from 2025](https://archive.org/details/radiance-geojson-data-2025), this project maps locations in the United States to their light pollution levels. Each layer of acrylic corresponds to a quartile on the polluted scale for this region, the least polluted layer visible at the top. Points are grouped using DBSCAN clustering into organic shapes that are cut away from each layer to reveal the light underneath. The areas with the highest levels of light pollution are visible from how much light shines through.
